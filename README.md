# Cats vs Dogs Image Classification using SVM

## Overview

This project implements a Support Vector Machine (SVM) model to classify images of cats and dogs from the Kaggle Cats and Dogs dataset. The images are preprocessed by resizing and converting them to grayscale before being used for training and testing.

The SVM algorithm learns patterns from the image data and predicts whether a given image belongs to a cat or a dog. This project demonstrates the application of machine learning and computer vision techniques for image classification tasks.

## Technologies Used

* Python
* OpenCV
* NumPy
* Scikit-Learn

## Features

* Image preprocessing
* Dataset loading and labeling
* SVM model training
* Accuracy evaluation
* Single image prediction

## Applications

* Animal image classification
* Computer vision projects
* Machine learning practice
* Image recognition systems

## Conclusion

This project showcases how Support Vector Machines can be used for binary image classification tasks. It provides a strong foundation for understanding image preprocessing, feature extraction, and machine learning model development.
